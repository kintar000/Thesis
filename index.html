Important Information

This is a static HTML file solution that allows browsing the SRPH-MIS application files directly without needing to run the Node.js server.

Note that some functionality may be limited when accessing the files directly:

  • Database operations will not work
  • API endpoints will not be available
  • Authentication may not function properly

This approach is useful when you can't run the Node.js server due to socket binding or other Windows Server limitations.

IIS Configuration

If you want to serve these files through IIS:

  1. Set up an IIS website pointing to the SRPH-MIS root directory
  2. Enable directory browsing in IIS
  3. Make sure default document includes "index.html"
  4. Set appropriate permissions for the IIS_IUSRS account

This will allow accessing the application through IIS without running Node.js.
