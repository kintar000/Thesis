1. SYSTEM OVERVIEW & ARCHITECTURE
--------------------------------------------------------------------------------

WHAT IS SRPH-MIS?
SRPH-MIS (San Roque Philippine Hospital - Management Information System) is an
enterprise-grade inventory and asset management platform designed for healthcare
and educational institutions.

CORE CAPABILITIES:
‚úì Complete IT asset lifecycle management
‚úì Multi-cloud infrastructure tracking (Azure, AWS, GCP)
‚úì Real-time virtual machine monitoring
‚úì Network device discovery and management
‚úì Software license compliance
‚úì Security and encryption features
‚úì Comprehensive audit trails
‚úì Role-based access control

TECHNOLOGY STACK:
- Frontend: React 18+ with TypeScript
- UI Framework: Tailwind CSS + shadcn/ui components
- Backend: Node.js with Express.js
- Database: PostgreSQL (production) / In-memory (development)
- Authentication: Session-based with MFA support
- Monitoring: Zabbix integration
- Encryption: AES-256-GCM

DEPLOYMENT OPTIONS:
- Replit Cloud Deployment (Recommended)
- Local Windows/Linux Server
- Port Configuration: 5000 (default, production-ready)

SYSTEM REQUIREMENTS:
Minimum:
- Node.js 18+ or 20+
- 2GB RAM
- 10GB disk space
- Modern web browser (Chrome, Firefox, Edge)

Recommended:
- Node.js 20 LTS
- 4GB+ RAM
- 20GB+ disk space
- PostgreSQL 12+

29. TROUBLESHOOTING GUIDE

COMMON ISSUES & SOLUTIONS:

AUTHENTICATION ISSUES:

1. Cannot Login:
   Symptoms:
   - Invalid credentials error
   - Login button not responding
   - Redirect loop

   Solutions:
   A. Verify credentials:
      - Check username spelling
      - Verify caps lock is off
      - Ensure correct password
      - Try password reset

   B. Clear browser data:
      - Clear cookies
      - Clear cache
      - Try incognito/private mode
      - Try different browser

   C. Check account status:
      - Account may be locked
      - Account may be inactive
      - Contact administrator

   D. System issues:
      - Check server status
      - Verify database connection
      - Check logs for errors

2. Account Locked:
   Symptoms:
   - "Account locked" message
   - Cannot login after multiple attempts

   Solutions:
   - Wait for lockout duration (default: 30 min)
   - Contact administrator to unlock
   - Admin: User Management ‚Üí Unlock Account
   - Check for suspicious activity

3. MFA Issues:
   Symptoms:
   - MFA code not accepted
   - Cannot scan QR code
   - Lost MFA device

   Solutions:
   A. Code not working:
      - Verify device time is correct
      - Try next code (wait 30 seconds)
      - Check for typos

   B. QR code issues:
      - Use manual entry code
      - Try different authenticator app
      - Refresh setup page

   C. Lost device:
      - Contact administrator
      - Admin can disable MFA temporarily
      - Re-setup MFA on new device
      - Use backup codes (if available)

4. Session Expires Quickly:
   Symptoms:
   - Frequent logout prompts
   - "Session expired" messages

   Solutions:
   - Check timeout settings (Admin ‚Üí System Setup)
   - Verify cookies are enabled
   - Avoid opening multiple tabs
   - Request timeout adjustment
   - Check browser settings

1. Connection Failed:
   Symptoms:
   - "Failed to connect to database"

   - Timeout errors

   C. Replit Database:
      - Check Replit Database tab

      - Re-copy connection string
      - Restart application

   D. Network:
      - Check internet connection
      - Verify firewall rules
      - Test database host reachability

2. Slow Performance:
   Symptoms:
   - Slow page loads
